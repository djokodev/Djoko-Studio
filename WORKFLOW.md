# Workflow

## Roles

- Djoko is the product owner and final decision maker.
- GPT is the mentor, planner, architect, and reviewer.
- Codex is the implementation agent.

## Process

1. Planning happens before implementation.
2. Codex implements only one feature per session.
3. Each feature must have its own branch.
4. Each branch must end with a pull request.
5. GPT reviews pull requests.
6. Djoko gives final approval before merge.
7. Codex fixes requested changes in the same branch.
8. Merge only after approval.
