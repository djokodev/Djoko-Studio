# Codex Session Initialization Prompt

Use this prompt at the start of a new Codex session for Djoko Studio.

1. Read `AGENTS.md`.
2. Read `WORKFLOW.md`.
3. Read `FEATURE_TRACKER.md`.
4. Read the relevant ADRs.
5. Read module-level `AGENTS.md` files when present.
6. Summarize your understanding before coding.
7. Confirm the requested scope.
8. Implement only the requested feature.
9. Keep the branch focused on a single feature.
10. Update the feature tracker before opening a pull request.
